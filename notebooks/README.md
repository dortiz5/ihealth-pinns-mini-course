# Notebooks para actividades computacionales con PINNs

Este repositorio contiene una serie de notebooks introductorios sobre
**Physics-Informed Neural Networks (PINNs)**, organizados de forma progresiva.

## Lecciones

- **Lect 1 - PINN Mass - spring- damper.ipynb**  
  *Introducción a PINNs con sistema masa–resorte–amortiguador.*

- **Lect 2 - ANN Vs PINN - pendulum.ipynb**  
  *Comparación entre ANN y PINN en un péndulo no lineal.*

- **Lect 3 - 2D PINNS diffusion - direct.ipynb**  
  *Resolución directa de la ecuación de difusión 2D.*

- **Lect 4 - 2D PINNS diffusion - inverse copy.ipynb**  
  *Estimación del coeficiente de difusión mediante PINNs.*
